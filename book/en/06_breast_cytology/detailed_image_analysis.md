# Unit 6: Detailed Image Analysis of Breast Cytology

*(Translation pending / Traducción pendiente)*

This chapter covers the anatomical and histological features of the breast, cytology examination methods (FNAC, imprint, nipple discharge, core needle biopsy), normal cytology patterns (epithelial, myoepithelial, stromal bipolar naked nuclei, apocrine metaplasia), non-tumoral pathology (mastitis, fat necrosis), and benign and malignant breast tumors (fibroadenoma, ductal carcinoma, lobular carcinoma).
